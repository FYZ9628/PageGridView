# VpGridView
ViewPager+GridView组合控件实现ViewPager+GridView实现网格布局分页效果

## VpGridView自定义属性：
属性名 | 说明 | 默认值
:----------- | :----------- | :-----------
pageSize                | 每页大小               | 8
numColumns              | 列数                   | 4
isShowIndicator         | 是否显示指示器          | true
selectedIndicator       | 选中指示点资源ID        | R.mipmap.ic_dot_selected
unSelectedIndicator     | 未选中指示点资源ID      | R.mipmap.ic_dot_normal
